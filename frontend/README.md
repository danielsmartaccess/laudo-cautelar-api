# Laudo Cautelar – Checklist Pro (Frontend)
App estático (HTML/CSS/JS) para preenchimento de laudo cautelar automotivo:
- Checklist completo por seções
- Cálculo de **IPA Score** (0–100) com badges de risco
- Upload e preview de fotos (local)
- Salvar/Carregar via **LocalStorage**
- Exportar/Importar JSON
- Resumo pronto para **impressão/PDF**

## Como rodar
1. Extraia o ZIP.
2. Abra o `index.html` no seu navegador (ou use Live Server no VS Code).
3. Preencha o formulário, clique **Calcular IPA** e **Gerar Resumo** para imprimir em PDF.

> App é 100% offline. Nenhum dado sai do seu computador.
